'this' Key word in Javascript

1. this is a keyword in Javascript.

2. this indicates to current object or refers to the object it belongs

3. In Javascript we have different ways to use 'this' keyword.

1st Way of using 'this'

a. 'this' at global level.

this.globalTest = 'global test';

Explanation:

1. if we use above statement inside a script directly then it act as global object.

2. Here in Javascript global object means window object.

3. So when we want to access such variables we can use (window.globalTest)

2nd Way of using 'this'

a. 'this' inside an object.

let objectLevel = {
objectName: 'sample object';
accessThis() {
console.log(this.objectName);
}
}

Explanation:

1. From above example if we want to access the object name we can't access using this.objectLevel.objectName.Because object name belongs to the ObjectLevel but not at window level.

2. If we want to access object name out of the objectLevel we can use in direct way objectLevel.objectName.

3. For example if we have a function inside the objectLevel i.e.. accessThis() we can access the objectName inside the function using 'this' keyword. we can access the function using objectLevel.accessThis()

3rd Way of using 'this'

a. 'this' inside a Function.

const windowLevelSample = "window level variable"; const functionLevel = function() {
console.log(this.windowLevelSample);
}
}

Explanation:

1. In the above example when we are trying to access the functionLevel() method it gives error i.e.. cant find 'this'. Because inside a function we cant access the global level variables directly.

2. To overcome this problem one of the method to use is 'call' method. See how to use call method in next level.

4th Way of using 'this'

a. 'this' inside a inner function.

const windowLevelSample = "window level variable"; const functionLevel = function() {
let this = this; // 2nd method const innerFunction = function() {
console.log(this.windowLevelSample);
}
innerFunction.call(this); // one method }

Explanation:

1. In the above example when we are trying to access 'this' inside a inner function it gives an error. Even when we pass 'this' from call method to a parent we can't access inside the inner function so this can be resolved in 3 ways.

2. We can all the inner function using 'call' method. innerFunction.call(this);

3. Create a variable for outer function and assign this as a value. now we can use the variable inside the inner function.

4. If we use inner function as arrow function we can directly access this keyword inside the inner function.

5th Way of using 'this'

a. 'this' using classes.

class sampleClass() {
constructor() {
this.sample =' sample class level name'; }
}

Explanation:

1. We can access the variables of a class inside the functions using 'this' keyword directly.

'let' Key word in Javascript

1. 'let' is a keyword in Javascript.

2. It is a block level scope in Javascript i.e.. when we are declare a let variable inside the 'if' block we can access them only inside the if block.

3.Hoisting will not happened for let variables

4. if we have a variable which have both inner and outer with same 'let' variables then inside the inner function we can always access the inner 'let' variable only
